Professional Background :office:

I'm currently based in Brighton, but I've spent time living in London too. I'm a software engineer working remotely for the Belfast office of Applied Systems, who are the market leaders for cloud based insurance management software. For a jargon-free example of what that actually means, the main project I work on is a set of .NET Core microservices that calculate insurance premiums in real time according to pricing models specified by insurers.

At Applied I predominantly work with C#, .NET Core and SQL Server on the backend, although I also work on internal web tooling using ASP.NET Core, JavaScript and HTML/CSS, and have previously made brief use of Golang and PowerShell to write internal CLI tools. So the day to day can vary.

Prior to Applied Systems I worked as a statistical analyst at a UK-based motor insurance company called First Central. My role there aligned closer with that of a data or research scientist, where freedom was given for teams to work in a research and development driven environment. The end goal was insurance pricing optimisation using machine learning techniques. Think along the lines of principal component analysis, clustering, gradient boosting and other statistical methods, all used to extract signal from streams of noisy data. Tech used was mainly the Python data science stack (NumPy, SciPy, Matplotlib, Pandas, etc.), SAS and SQL.
